### Integração Meus Pedidos A integração com o Meus Pedidos é feita através de uma API em protocolo HTTP (modelo REST), e os dados são comunicados em formato JSON. Apesar das siglas complicadas, o processo é muito simples. Seu software poderá se comunicar com nossos servidores através de requisições HTTP, de forma muito parecida com um navegador de internet. O protocolo HTTP é utilizado pelos navegadores para se comunicar com servidores web, através dos métodos GET e POST. Você irá utilizar estes mesmos métodos para fazer leitura, cadastro e atualização de registros em nossa API. Utilizamos o protocolo HTTP em ambiente de teste, no ambiente produção são utilizadas requisições HTTPS. É possível desenvolver a integração em qualquer linguagem de programação. Os únicos componentes que você irá precisar são: uma biblioteca para realizar conexões HTTP, e uma biblioteca para leitura/escrita de dados em formato JSON. A maioria das linguagens possuem bibliotecas prontas para este tipo de operação.
